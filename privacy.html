Privacy Policy

Effective Date: July 26, 2025

This application ("the App") is a private tool used only by the Developer for uploading videos to their own TikTok account.

What Data Is Collected

  • TikTok OAuth access tokens
  • Video file metadata (e.g., file name, caption)

How the Data Is Used

  • The access token is used solely to authenticate API requests to the TikTok platform.
  • Metadata is used to properly post the content to TikTok as intended by the Developer.

Data Sharing

  • No data is shared with third parties.
  • The App does not collect or transmit any data unrelated to the posting process.
  • The App does not support or allow access by other users.

Data Retention

  • OAuth tokens may be stored securely by the Developer for continued use.
  • No user data (since there are no users) is stored or retained long-term.

For questions or concerns about this policy, please contact your@email.com.
